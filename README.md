# SentinelPath v1.0 | Open-Source Web Auditor

## Mission
To bridge the gap between web development and cybersecurity education by providing real-time heuristic analysis of client-side vulnerabilities.

## Technical Architecture
- **Engine:** JavaScript (ES6+) with Chrome Extensions API v3.
- **Security Logic:** Implements pattern recognition for insecure data transmission and DOM-based anomalies.
- **Focus:** Preventing MITM (Man-in-the-Middle) attacks by flagging unencrypted login vectors.

## Why this project?
Developed as part of a research project into modern web security standards.
